class seamm_ase.seamm_ase.SEAMM_Calculator(step, calculator=None, name=None, configuration=None, **kwargs)[source]#

Bases: Calculator

Generic ASE calculator for SEAMM.

This is a generic calculator that can be used from any step in SEAMM to use functionality in ASE.

The step must have a calculator method that is called by this class:

def calculator(
    self,
    calculator,
    properties=["energy"],
    system_changes=ASE_all_changes,
):
    """Create a calculator for the structure step.

    Parameters
    ----------
    ase : ase.calculators.calculator.Calculator
        The ASE calculator we are working for
    properties : list of str
        The properties to calculate.
    system_changes : int
        The changes to the system.
    """
...

An example can be found in the Structure step.

The step must also create the SEAMM_Calculator, passing itself into the constructor, and set up the Atoms object to use this calculator:

...
symbols = configuration.atoms.symbols
XYZ = configuration.atoms.coordinates

calculator = SEAMM_Calculator(self)
atoms = ASE_Atoms("".join(symbols), positions=XYZ, calculator=calculator)
...

The step can then call the calculate method of the SEAMM_Calculator to perform the calculation, or can pass the calculator to other ASE drivers that will use the calculator.
